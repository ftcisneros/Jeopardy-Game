How to play Jeopardy!

Objective: Answer questions from various categories to earn points. The player or team with the highest score at the end wins.

Game Setup

Categories: Game board has multiple categories, each with questions of increasing point values.

Gameplay

Starting the Game:

Players select a category and question value to begin.

Answering Questions:

Players have a limited time to respond.

Answers must be phrased as questions (e.g., "What is..." or "Who is...").

Scoring:

Correct Answer: Points for the question are added to the player's score.

Incorrect Answer: Points are deducted, and the question is marked “ANSWERED.”

Player Turns (Multiplayer Mode):

Current player's turn is highlighted.

Incorrect answers pass the turn to the next player.

End of Game

Winning: Player with the highest score at the end wins.

Leaderboard: Displays player rankings based on final scores.
